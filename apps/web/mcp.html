Model Context Protocol

AI asistanlar için Base cüzdan muhasebesi.

TxReceipts MCP, AI asistanların (Claude, Cursor, VS Code Copilot, ve diğer MCP istemcileri) Base cüzdan aktivitenizi okumasını, işlem özetlerini almasını ve ön-muhasebe raporları hazırlamasını sağlar. Tüm araçlar salt okunurdur — asistanınız cüzdanınıza dokunamaz, onay isteyemez veya transfer yapamaz.

Nasıl çalışır

MCP = AI asistanınıza bir veri kanalı

MCP (Model Context Protocol), AI asistanlarının harici araçları ve veri kaynaklarını kullanması için standart bir protokoldür. TxReceipts MCP ile asistanınız doğrudan Base blokzincirinden cüzdan aktivitenizi okuyabilir.

1
MCP istemcinize sunucuyu ekleyin

VS Code, Claude Desktop, Cursor veya herhangi bir MCP istemcisine TxReceipts MCP sunucu adresini yapılandırın.

2
Asistanınıza sorun

"Base cüzdanımın son 10 işlemini göster" ya da "Bu ay ne kadar USDC harcadım?" gibi sorular sorun.

3
Ön-muhasebe raporu alın

Asistan, TxReceipts MCP araçlarını kullanarak cüzdan aktivitenizi analiz eder ve muhasebeye hazır bir özet sunar.

MCP Araçları

Salt okunur muhasebe araçları

TxReceipts MCP sunucusu 6 araç sunar. Tümü salt okunurdur — imzalama, transfer, onay veya yazma işlemi yapmaz.

Araç 1

list_networks

Desteklenen ağları listeler (Base, Ethereum, Optimism, Arbitrum, Polygon). Her ağın RPC URL, gezgin adresi ve yeteneklerini döndürür.

Araç 2

get_wallet_activity

Bir cüzdanın işlem geçmişini ve token transferlerini getirir. Ağ, cüzdan adresi ve limit parametrelerini alır.

Araç 3

get_transaction_summary

Tek bir işlemin detaylı özetini döndürür: gönderilen değer, gas ücreti, token hareketleri, blok zamanı ve durum.

Araç 4

get_public_receipt

Geçerli bir TXR-XXXX-XXXX makbuz ID'sine göre herkese açık makbuzu getirir.

Araç 5

create_receipt

API anahtarı ile muhasebe makbuzu oluşturur. Kimlik doğrulama gerektirir, sadece yetkili projeler kullanabilir.

Araç 6

get_pre_accounting_report

Bir cüzdan için kapsamlı ön-muhasebe raporu hazırlar. Token akışlarını, gaz harcamalarını, gelen/giden özetini ve ağ yeteneklerini içerir. Base ağında full yetenek desteği ile çalışır.

Kurulum

MCP istemcinize ekleyin

VS Code (settings.json) 
{
  "mcp": {
    "servers": {
      "txreceipts": {
        "type": "url",
        "url": "https://txreceipts-api.evpc77.workers.dev/mcp"
      }
    }
  }
}

VS Code ayarlarınıza ekleyin, Copilot Chat otomatik olarak TxReceipts MCP araçlarını kullanabilir.

Claude Desktop (claude_desktop_config.json) 
{
  "mcpServers": {
    "txreceipts": {
      "type": "url",
      "url": "https://txreceipts-api.evpc77.workers.dev/mcp"
    }
  }
}

Yapılandırmayı ekleyip Claude'u yeniden başlatın. Doğrudan Base cüzdan soruları sorabilirsiniz.

MCP Keşif Uç Noktası 
URL: https://txreceipts-api.evpc77.workers.dev/.well-known/mcp.json

Bu uç nokta, MCP keşif araçları tarafından taranabilir ve TxReceipts'in Base ekosisteminde bulunabilirliğini sağlar.

Güvenlik

Salt okunur. Onay yok. Transfer yok.

  • TxReceipts MCP salt okunurdur — imzalama, onay, transfer veya swap işlemi yapmaz
  • Hiçbir özel anahtar, seed phrase veya API anahtarı MCP bağlamına gönderilmez
  • Cüzdan bakiyeleri ve işlemler yalnızca genel blokzincir verilerinden okunur
  • create_receipt aracı ek bir API anahtarı gerektirir ve isteğe bağlıdır
  • Her MCP isteği için hız sınırı uygulanır
  • AI asistanı, sizin izniniz olmadan MCP aracını çağıramaz

SSS

Sık sorulan sorular

MCP hangi AI asistanları ile çalışır?

Claude Desktop, VS Code Copilot Chat, Cursor, GitHub Copilot ve MCP protokolünü destekleyen tüm istemciler.

MCP'yi kullanmak için ücret ödemem gerekir mi?

Hayır. MCP erişimi tamamen ücretsizdir ve bir API anahtarı gerektirmez. Sadece MCP sunucu URL'sini istemcinize eklemeniz yeterlidir.

Asistanım cüzdanımdan token transferi yapabilir mi?

Hayır. Tüm MCP araçları salt okunurdur. Asistanınız blokzincir verilerini okuyabilir ancak imzalama, onay veya transfer işlemi yapamaz.

Hangi ağlar destekleniyor?

Base (varsayılan), Ethereum, Optimism, Arbitrum ve Polygon. Base ağı full yetenek desteği ile çalışır (MCP, X402, Paymaster, Base Account, Builder Codes).

Ön-muhasebe raporu resmi bir muhasebe kaydı mıdır?

Hayır. Rapor bir ön-muhasebe kaydıdır ve resmi bir fatura, vergi beyannamesi veya e-defter kaydı değildir. Resmi muhasebe için lisanslı bir muhasebeciye danışın.
