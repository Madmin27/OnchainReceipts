Base USDC credits

Prepaid credits keep billing simple for teams.

Dapps can top up paid API requests with native USDC on Base. The launch conversion is designed for low-friction integration and clear reconciliation.

  • Every project gets the first 1,000 API requests free
  • 1 USDC equals 2,000 additional API requests
  • Minimum top-up is 5 USDC
  • Duplicate project, chain, and tx hash requests are not double counted
  • User-generated receipts from the public app stay free at launch
